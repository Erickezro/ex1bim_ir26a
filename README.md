# ex1bim_ir26a

Repositorio para el Examen 1er Bimestre - IR26A

## Descripción

Este repositorio contiene los notebooks desarrollados para el primer examen del bimestre de la materia Recuperación de Información. Todo el contenido se encuentra en formato **Jupyter Notebook**.

## Requisitos

- [Python 3.x](https://www.python.org/downloads/)
- [Jupyter Notebook](https://jupyter.org/install)
- Paquetes adicionales (instalar con pip si el notebook lo requiere)

## Instrucciones de uso

1. **Clona este repositorio:**

   ```bash
   git clone https://github.com/Erickezro/ex1bim_ir26a.git
   cd ex1bim_ir26a
   ```

2. **(Opcional) Crea un entorno virtual:**

   ```bash
   python -m venv venv
   source venv/bin/activate  # En Windows: venv\Scripts\activate
   ```

3. **Instala Jupyter Notebook y dependencias:**

   Se tiene un archivo `requirements.txt`, ejecuta:

   ```bash
   pip install -r requirements.txt
   ```

4. **Inicia Jupyter Notebook:**

   ```bash
   jupyter notebook
   ```

5. **Abre el notebook deseado desde el navegador y sigue las instrucciones dentro del notebook.**
